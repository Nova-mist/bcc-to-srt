## 概念

![blog-image-20211004-1633357753](D:\Desktop\Rename\blog-image-20211004-1633357753.jpg)

**SRT字幕格式：**

>   字幕序号
>   字幕显示的起止时间
>   字幕内容（可多行）
>   空白行（表示本字幕段的结束）

**也可以设置字幕内容的字体格式。**

```
<font color=red>颜色</font>
<i>字体斜体</i>
<u>字体下加划线</u>
<br>换行
<b>字体加粗</b>
```

## 使用

1.   **将 json 文件拖入到批处理文件。**
2.   **在 srt 文件夹中会生成 srt 文件。**
